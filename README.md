# Jersey Shop - Full Stack E-Commerce Application

A complete jersey e-commerce platform built with React, Express.js, and Paystack payment integration.

## Features

- 🛍️ Browse football jerseys (Club & National teams)
- 🔍 Search and filter products
- 🛒 Shopping cart with size selection
- 💳 Paystack payment integration (GHS currency)
- 📦 Multiple delivery options (Standard, Express, Personal Rider)
- 📱 Fully responsive design
- ⚡ Real-time cart updates

## Tech Stack

### Frontend
- React 18
- Vite
- React Router DOM
- Axios
- React Icons
- React Hot Toast
- Paystack React Pay

### Backend
- Node.js
- Express.js
- MongoDB (optional)
- JWT Authentication
- Paystack API
